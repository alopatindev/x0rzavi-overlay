# x0rzavi-overlay
Gentoo overlay for my absolutely necessary utilities :P

## Add overlay
1. ```eselect repository add x0rzavi git https://github.com/x0rzavi/x0rzavi-overlay.git```
2. ```emaint sync -r x0rzavi```
